# 安雅健康产业 APP (HTML 原型)

这是一个基于 HTML/CSS 的移动端 APP 原型项目，展示了安雅健康产业 APP 的主要功能页面和交互流程。

## 📱 在线预览

**点击下方链接直接体验：**
> **[👉 在线预览 (GitHub Pages)](https://Misaki-hwen.github.io/anya-health-app/)**

*(注：如果您刚刚开启 GitHub Pages，可能需要等待 1-2 分钟链接才会生效)*

## 📑 包含页面

| 页面 | 说明 |
| :--- | :--- |
| **登录页** | 模拟手机号登录流程 |
| **首页** | 资产展示、快捷入口、公告 |
| **商城** | 商品列表、会员折扣展示 |
| **商品详情** | 商品信息、下单模拟 |
| **钱包** | 积分/贡献值资产管理、转账功能 |
| **交易窗口** | 贡献积分买卖盘口 |
| **我的团队** | 邀请码、团队数据统计 |
| **个人中心** | 基础设置入口 |

## 🛠️ 如何使用

1. 打开 [在线预览链接](https://Misaki-hwen.github.io/anya-health-app/)。
2. 使用浏览器（推荐手机模式或手机浏览器）访问。
3. 点击页面上的按钮进行交互跳转。

## 🚀 如何开启在线预览 (GitHub Pages)

如果您是仓库管理员，请按照以下步骤开启免费托管：

1. 进入 GitHub 仓库页面。
2. 点击顶部的 **Settings** (设置) 选项卡。
3. 在左侧菜单找到 **Pages**。
4. 在 **Build and deployment** > **Branch** 下：
   - 选择 `main` 分支。
   - 文件夹选择 `/ (root)`。
   - 点击 **Save**。
5. 等待页面顶部出现绿色的 URL 链接即可分享给他人。
